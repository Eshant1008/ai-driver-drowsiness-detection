OBJECTIVE:
To detect driver fatigue using computer vision and prevent accidents.

WORKING PRINCIPLE:
The system captures live video using webcam.
It detects facial landmarks using Mediapipe Face Mesh.
Eye Aspect Ratio (EAR) is calculated.

If EAR falls below threshold for continuous frames,
the system assumes the driver is sleeping and triggers alarm.

SCIENTIFIC CONCEPT:
Normal human blink lasts 0.1–0.4 seconds.
Long eye closure indicates microsleep.
EAR helps measure eye openness mathematically.
